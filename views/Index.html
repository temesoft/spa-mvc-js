spa-mvc-js

        The goal of this project is create a simple, structured single page application model-view-controller framework for JavaScript running inside the browser. The main idea is separation of view, controller and model within the application. Remote calls and controller logic should be separate from views, layouts and styles. 
        Features:
        - Encrypted hash parameters option to turn this parametrized hash url:
        /spa-mvc-js/#overview|1|2|Thu Apr 24 2014 22:35:50 GMT-0700 (PDT)|4|3214|00....
                into this:
        /spa-mvc-js/#overview|U2FsdGVkX1/9clPKqoXzH8OhefejtUKr1k9uD1VJgwKIfTtQmvqZQx+oBGxxYHdKHYhahcwyLa/286X......

Web root location structure:

/ {root}
|
|___/js {dir}
|   |
|   |___/controllers {dir: with java script controller classes files}
|   |
|   |___/context.js {file: initial mvc context}
|
|___/views {dir: with pure html files}
|
|___/index.html {file: initial (single) page template}
