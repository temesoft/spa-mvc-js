spa-mvc-js

        This is yet, another project trying to define a simple, structured, single page application model-view-controller framework for JavaScript running inside the browser. The main idea is separation of view, controller and model within the application.

Web root location structure:

/ {root}
|
|___/js {dir}
|   |
|   |___/controllers {dir: with java script controller classes files}
|   |
|   |___/views {dir: with pure html files}
|   |
|   |___/context.js {file: initial mvc context}
|
|___/index.html {file: initial page template}
